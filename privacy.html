一、隱私權保護政策的適用範圍

隱私權保護政策內容，包括本網站如何處理在您使用網站服務時收集到的個人識別資料。

隱私權保護政策不適用於本網站以外的相關連結網站，也不適用於非本網站所委託或參與管理的人員。

二、資料的蒐集與使用方式

本網站不主動收集您的個人識別資料。

三、資料之保護

本網站不收集您的個人識別資料。本網站架設於 GitHub Pages 平台，主機端的資料安全防護措施由 GitHub 負責，僅由經過授權的人員才能接觸與處理相關資料。

四、GitHub Pages 平台說明

本網站架設於 GitHub Pages。GitHub 可能依其服務條款，自動記錄存取伺服器的相關日誌（如 IP 位址、瀏覽器類型、存取時間等），此部分資料由 GitHub 依其隱私權政策處理，與本網站無關。

五、網站對外的相關連結

本網站網頁提供其他網站的網路連結，您也可經由本網站所提供的連結，點選進入其他網站。但該連結網站中不適用本網站的隱私權保護政策，您必須參考該連結網站中的隱私權保護政策。

六、與第三人共用個人資料之政策

本網站不收集您的個人識別資料。因此不會提供、交換、出租或出售任何您的個人資料給其他個人、團體、私人企業或公務機關。

七、Cookie 之使用

本網站不使用 Cookie。

八、Web Storage 之使用

為了提供您最佳的服務，本網站會在您的裝置中存取 Web Storage（localStorage），用於儲存您的瀏覽偏好設定（如自訂車種色系）及免責聲明確認狀態。若您不願接受 Web Storage 的寫入，可在瀏覽器設定中停用，但可能導致網站部分功能無法正常運作。

九、隱私權保護政策之修正

本網站隱私權保護政策將因應需求隨時進行修正，修正後的條款將刊登於網站上。
